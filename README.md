# FinanceTracker

A personal finance tracking app built with React Native (Android).

## Requirements

- Node.js >= 22.11.0
- Android device or emulator connected via USB
- Java 17 (JDK)

## Running

```powershell
npm run android
```

This starts Metro, wires up hot reload over USB, builds and installs the debug APK, then launches the app. After the first run, saving any `.ts`/`.tsx` file refreshes the app instantly — no rebuild needed.

## Tech

- React Native 0.85 (New Architecture)
- React Navigation
- Reanimated + Gesture Handler
- AsyncStorage
